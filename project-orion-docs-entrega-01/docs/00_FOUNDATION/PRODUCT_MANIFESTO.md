## Manifesto

Nós acreditamos que fidelidade não se compra apenas com desconto.

Fidelidade nasce quando o cliente sente que a empresa o reconhece, entende sua rotina, antecipa suas necessidades e oferece valor no momento certo.

A maioria dos varejistas possui dados valiosos, mas não consegue transformá-los em ação. Project ORION existe para transformar histórico de compras em relacionamento inteligente.

## Promessa do produto

Ajudar empresas a venderem novamente para quem já confia nelas.

## Experiência desejada

Para o lojista, deve parecer que existe um analista comercial inteligente trabalhando todos os dias. Para o cliente final, deve parecer que a loja o conhece de verdade.

## AI Instructions

Ao desenvolver funcionalidades, avalie se elas aumentam retenção, personalização, inteligência ou recorrência.
